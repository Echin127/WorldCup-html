# Socceroos Qualifying Journey
Simple html, mapping out the qualification journey of the Socceroos for the 2026 World Cup. It's an interactive site with the ability to move between locations chronologically.

Locations obtained through the ESPN API and assembled into a geojson file. Then, the venue details are combined with a Leaflet-based OpenStreetMap to create the final site. Written with the assistance of Claude Code.
